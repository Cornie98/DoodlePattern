# DoodlePattern ✎

A playful, local seamless pattern maker. Draw on a square tile, wrap the canvas to bring edges into the center, and export a PNG.

## Tech stack

- **Vite** - build tool and dev server
- **React 19** - UI
- **TypeScript** - typed app code
- **HTML Canvas 2D** - drawing, wrap, stamps, and PNG export




## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## How to use

1. Draw with pencil, marker, brush, eraser, fill, or stickers.
2. Click **Patternise** to shift the tile by 50% so seams meet in the middle.
3. Draw over the seams.
4. Toggle **Pattern preview** to see infinite repeat.
5. **Export PNG** to download the tile to your device.
